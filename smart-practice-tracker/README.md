# PracticeLog 🎸
*A smart music practice tracker built for musicians who want consistency, discipline, and measurable progress.*

🔗 **Live Demo:**  
https://logwithus.vercel.app

---

## Overview

PracticeLog is a full-stack web application that helps musicians track practice sessions, build streaks, analyze progress, and stay motivated through achievements and goals.

The app combines session logging, analytics, heatmaps, streak systems, and goal tracking into a clean and responsive interface designed specifically for daily music practice.

Built using Vanilla JavaScript, HTML/CSS, Node.js, and MySQL.

---

## Features

- Secure authentication with email verification & password reset  
- Practice timer with live session tracking  
- Session logging with mood, notes, focus area & instruments  
- Daily streaks and achievement system  
- GitHub-style activity heatmaps  
- Weekly and monthly analytics charts  
- Custom practice goals and progress tracking  
- Responsive UI for desktop and mobile devices

## Why I Built This

As a musician myself, I wanted a practice tracker that felt motivating instead of boring. Most existing apps either focus too much on productivity or feel overly complicated. PracticeLog was built to make daily practice more visual, rewarding, and consistent for musicians.

# Atharva Vaidya Portfolio

A futuristic cyberpunk-inspired portfolio website showcasing my journey as a Software Engineer, Cybersecurity student, and musician.

## Features

* Interactive particle network background
* Typewriter hero animation
* Smooth scroll reveal effects
* Responsive design
* Modern cyberpunk UI
* Project showcase
* Contact form interactions

## Tech Stack

* HTML
* CSS
* JavaScript (ES6+)
* Vite

## Projects Featured

* PracticeLog by LogWithUs
* Guitar Chord Library (In Development)
* PracticeFlow (Planned)

## Run Locally

```bash
npm install
npm run dev
```
